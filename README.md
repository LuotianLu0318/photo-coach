# 📸 拍照引导神器（不被骂的拍照神器）

举着手机取景时，屏幕**实时**告诉你怎么调，让原图在按快门前就已经**构图正、人好看、光线对**。纯前端 PWA，全程手机本地运算，无后端、不上传任何画面。

## 它会实时帮你看什么

- **构图**：水平线纠歪、三分网格、头顶留白、警告“切在关节处”、主体该往哪挪
- **姿势**：6 个姿势剪影模板贴在画面上照着摆、正脸→侧脸建议、收下巴、机位高低（显腿长）
- **光线**：逆光黑脸预警、脸偏暗/过曝提示、顶光阴影提醒、黄金时刻提示

提示按重要性排序，**每次最多 2 条**、带去抖不刷屏；全部达标时**快门变绿灯**。

## 怎么跑（关键：摄像头和传感器都要求 HTTPS 或 localhost）

### A. 本机预览（电脑上看界面，但手机摄像头体验要用 B/C）
```bash
cd "不被骂的拍照神器"
python3 -m http.server 8000
# 浏览器打开 http://localhost:8000
```
> 注意：手机连同一 Wi-Fi 访问电脑 IP（`http://192.168.x.x:8000`）时，iOS **不会**在非 HTTPS 下给摄像头/传感器权限。手机真机请用 C 方案（HTTPS）。

### B. 手机真机临时联调（HTTPS 隧道）
```bash
python3 -m http.server 8000
npx localtunnel --port 8000     # 或 cloudflared tunnel --url http://localhost:8000
# 用它给的 https 网址在手机上打开
```

### C. 长期使用（推荐）：托管到 GitHub Pages
1. 把本目录推到一个 GitHub 仓库
2. 仓库 Settings → Pages → 选 main 分支根目录，保存
3. 拿到 `https://<用户名>.github.io/<仓库>/` 网址，手机打开
4. Safari/Chrome → 分享 → **添加到主屏幕**，从此点图标即用

## 手机上使用
1. 打开网址 → 点「开始拍照」→ 允许**摄像头**和**运动与方向**权限（iOS 必须在这一步点允许）
2. 举起手机对人取景，跟着顶部提示调整；想给对方摆姿势就点左下角 🧍 选模板
3. 快门变绿 ✅ 时按下；拍后可「保存到相册」

## 已知与校准
- **水平线方向**：倾斜符号在不同机型可能相反；真机看一眼，如果转的方向反了，改 `app/overlay.js` 里 `ctx.rotate(a)` 为 `ctx.rotate(-a)` 即可。
- 首次加载会从 CDN 下载 AI 模型（约几 MB），需联网；之后浏览器会缓存。
- 存图为浏览器下载/长按保存；画质取自摄像头原生分辨率，叠加的构图线**不会**进照片。

## 不做（v1 范围）
后期调色/滤镜、账号云端、自制硬件、精确日照计算、背景杂乱检测——按需再加。

## 代码结构（`app/`）
`main.js` 主循环 · `camera.js` 取流/拍摄 · `vision.js` 人脸+骨架检测 · `sensors.js` 倾角/时间 ·
`light.js` 亮度分析 · `composition.js` 构图 · `pose.js` 姿势模板+角度 · `coach.js` 规则引擎 ·
`overlay.js` 画面叠加 · `ui.js` 提示与控件
